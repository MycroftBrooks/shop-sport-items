Магазин одежды на Django

Магазин одежды "AKE Store" был построен на фреймворке Django, языке Python. В магазине можно просматривать товары, добавлять их в корзину, оформлять заказы. Также была реализована система авторизации и регистрации пользователей, а также система администрирования, в которой можно добавлять, редактировать и удалять товары. Для работы с магазином необходимо зарегистрироваться или авторизоваться.

При создании магазина использовались следующие технологии:

  • Django 4.1.3
  • Python 3.11.0
  • Bootstrap 5.2.2
  • SQLite 3.39.4
  • HTML 5
  • CSS 3
